# Changelog

All notable changes to Vyra will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-24

### Added
- Initial release of Vyra
- Natural language programming syntax
- Core language features:
  - Variables and assignments
  - Arithmetic operations
  - Input/Output operations
  - Conditionals (if/else)
  - Loops (while, for-each, repeat N times)
  - Lists and basic data structures
  - File I/O operations
  - Comments
- Parser with pattern-based English recognition
- Logic Graph intermediate representation
- Interpreter engine for execution
- CLI with commands:
  - `run` - Execute programs
  - `repl` - Interactive REPL
  - `parse` - Parse and visualize
- Graph visualization support
- Comprehensive examples:
  - Hello World
  - Calculator
  - Number Guessing Game
  - Fibonacci Generator
  - List Processing
  - File I/O
  - Temperature Converter
- Full test suite
- Documentation:
  - Language Specification
  - Tutorial
  - Quick Reference
  - Contributing Guide
  - API Documentation

### Features
- Zero-syntax programming in plain English
- Deterministic execution via logic graphs
- Educational focus with clear error messages
- Extensible architecture for future enhancements
- Cross-platform support (Windows, Linux, Mac)

### Performance
- Efficient graph-based execution
- Minimal overhead for simple programs
- Iteration safety with configurable limits
